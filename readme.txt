# Test Yourself Maths

Test Yourself Maths is a web application designed to help users practice and improve their mathematics skills through interactive quizzes.

## Features

- Simple Multiplication Quiz (Times tables: 1 to 12)
- Simple Division Quiz (Factors: 1 to 12)
- Instant feedback on score out of 12 questions.
- Noise-free user-friendly interface

## Getting Started

1. Clone the repository:
    ```
    git clone https://github.com/crogers08/testyourselfmaths.git
    ```
2. Install dependencies:
    ```
    npm install
    ```
3. Start the development server:
    ```
    npm start
    ```

## Technologies Used

- HTML, CSS, JavaScript
- Bootstrap
- Node.js
